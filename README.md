# 🎮 Game Art Showcase: Zombie & Horror

**Проект Academy Top (Батайск, 2026) | 1-й курс**

Этот репозиторий содержит галерею игровой графики, созданной студентами первого курса Academy Top (Батайск) в рамках финального проекта. Тематика — зомби и хоррор.

## 📂 О проекте

Студенты разработали визуальные ассеты для двух игр:
*   **🧟 Zombie Game**
*   **👻 Horror Game**

Все изображения созданы с нуля в программе **GIMP** (GNU Image Manipulation Program). В галерее представлены:

*   🧩 Тайлы (окружение, уровни)
*   🌄 Фоны
*   🧠 Боссы
*   🧍 Главные герои
*   👥 NPC (дружественные персонажи)
*   🧟 Враги
*   🎒 Предметы (оружие, инвентарь)
*   ❤️ UI-элементы и эффекты (меню, индикаторы, анимации)

## 🚀 Как запустить

1.  Скачайте или клонируйте репозиторий.
2.  Откройте файл `index.html` в любом современном браузере.
3.  Выберите команду (Zombie или Horror), затем категорию артов, чтобы увидеть работы.

## 🛠 Технологии

*   HTML5
*   CSS3 (шрифт Press Start 2P)
*   JavaScript
*   **GIMP** — основной инструмент для создания графики
